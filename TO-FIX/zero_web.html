Progressive Web App

A website that thinks it's a mobile app and a mobile app that thinks it's a website, well that’s just genius.

Progressive web apps are a hybrid of two technologies; basically, a PWA is a website that gets downloaded to the device as an offline file, so you can still view content even when disconnected from wifi or mobile data. Furthermore, PWA is allowed access to some aspects of the device's hardware, which is helpful.

You need a website! Now what?

Throwing together a basic website is easy; you can do everything yourself online for just a few pounds and get up and running in under an hour. There are loads of web hosting companies out there, and they all offer drag and drop website creation in some form or another. If you have the patience and don't mind typing in the physical content, you will save a fair bit of money and can play around with layouts and styles to your heart's desire.

Most web developers and designers don't write lots of physical computer code; they use pre-made templates and make a few tweaks and changes to fit their client's requirements. This is not bad, as programmers are inherently lazy and repeatedly try to utilise the same piece of computer code. This saves time and is the most efficient way of writing. First, write something great, test it thoroughly, deploy, reuse, it's that simple. If you were building a house from scratch, you would not make individual bricks, windows or roof tiles; you would buy the best products and assemble them in the order you want; building websites use the same principles.

So we have said building websites is straightforward, and that is true if you are using a simple drag and drop template. Still, the real magic is the code that runs behind these systems or the physical coding that the web developer writes for themselves and any integrations or plugins you need. Websites are not written in just one computer language either; even our very basic websites utilise many different code types, HTML, SCSS, CSS, Java, PHP, JSON & MySQL, so unless you are prepared to spend some serious time learning how to write code from scratch the only choice you have is to use a WYSIWYG (What you see is what you get) drag and drop template or pay a proficient developer.
